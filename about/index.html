Sobre el sitio - Wed, Mar 29, 2023

Son mis notas. Mal hechas, pero tal vez te sirvan

GitHub te deja tener un sitio personal, así que lo voy a usar como vertedero para notas mal hechas de las cosas que siempre olvido en cuanto termino de usarlas.

También subiré notas de los mejores cursos que llevé en la universidad, pero a esas no tiene sentido darles un formato “de blog”, así que tendrán sus propias entradas con pdf’s para descargar.

Me interesa saber si encontraste este sitio, y cómo lo encontraste. Si puedes escribirme un correo, estaría muy bien.

Este sitio está hecho con HUGO, y usa el tema Hugo.i386 como base.

Si no logro hacer que este tema se lleve bien con KaTeX, volveré a usar Tokiwa, que está bastante más pulido y es relativamente limpio, pero no tiene el aspecto sólido y sin distracciones de una interfaz de terminal.
